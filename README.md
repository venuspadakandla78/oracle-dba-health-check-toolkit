# Oracle DBA Health Check Toolkit

A practical collection of Oracle DBA health check scripts used in real production environments to monitor database performance, availability, storage, backup, and security.

This toolkit is designed for Oracle DBAs, consultants, and cloud engineers who want quick visibility into database health and operational risks.

---

## Features

- Database availability & instance status checks
- Tablespace usage and growth monitoring
- Blocking sessions and locking analysis
- Top SQL and performance diagnostics
- RMAN backup status verification
- Data Guard lag and role validation
- ASM diskgroup utilization
- Security and audit verification queries

---

## Technologies Used

- Oracle SQL / PL-SQL
- Linux Shell Scripting
- Oracle AWR / ASH Views
- RMAN
- Oracle Data Guard
- ASM
